# 🏡 BC Imóveis

Projeto de catálogo de imóveis desenvolvido para a **BC Imóveis**, com foco em uma navegação intuitiva, design moderno e responsividade. A interface foi projetada no **Figma** com princípios de **UI/UX** e desenvolvida utilizando **HTML**, **CSS** e **JavaScript**.

## 📖 Sobre o projeto

Este projeto tem como objetivo apresentar um catálogo de imóveis de forma organizada, oferecendo uma experiência de navegação simples, rápida e agradável para os usuários.

## ✨ Funcionalidades

* Catálogo de imóveis
* Layout responsivo
* Interface moderna
* Navegação intuitiva
* Código organizado e de fácil manutenção

## 🛠️ Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript
* Figma (UI/UX)

## 📂 Estrutura do projeto

```text
bc-imoveis/
├── assets/
│   ├── css/
│   ├── js/
│   └── images/
├── index.html
└── README.md
```

## 🚀 Como executar

1. Clone o repositório:

```bash
git clone https://github.com/seu-usuario/bc-imoveis.git
```

2. Acesse a pasta do projeto:

```bash
cd bc-imoveis
```

3. Abra o arquivo `index.html` em seu navegador ou utilize uma extensão como **Live Server** no Visual Studio Code.
